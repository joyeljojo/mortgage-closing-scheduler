# Mortgage Closing Scheduler

## Overview

A Python-based mortgage closing scheduler that automates attorney assignment based on business rules used in mortgage closing operations.

## Features

- County-based attorney assignment
- Phone and physical closing handling
- Attorney fee tracking
- DNU (Do Not Use) attorney restrictions
- Attorney rotation system
- Borrower search functionality
- Closing statistics reporting
- Attorney utilization tracking
- Multi-state support

## Technologies Used

- Python
- Dictionaries
- Lists
- Functions
- Loops
- Conditional Logic

## Sample Functions

- Assign attorney
- Search borrower
- View closings
- Generate statistics
- Track attorney usage

## Future Improvements
